# Weather App

A weather app built with React, TypeScript, and the OpenWeatherMap API.

## Live Demo
[weather-app-eta-lovat.vercel.app](https://weather-app-eta-lovat.vercel.app)

## Features
- Search any city and get real-time weather data
- Autocomplete dropdown with multiple city results
- Save cities to a sidebar for quick access
- Saved cities persist across sessions with localStorage

## Tech Stack
- React + TypeScript
- Vite
- OpenWeatherMap API

## Run Locally
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with `VITE_WEATHER_API_KEY=your_key_here`
4. Run `npm run dev`
